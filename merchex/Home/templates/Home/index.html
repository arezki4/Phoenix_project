Pourquoi remplir le formulaire ?

La musique que vous écoutez peut vous influencer.
Elle peut drastiquement changer votre humeur.

Afin de vous aidez au mieux a contrôler votre humeur via la musique, nous vous prions de remplir les formulaires suivant, qui nous permettrons a l'avenir de vous proposer des playlist qui vous améliorerons votre mental.

Info

Nous somme un petit groupe de deux personnes qui essaye de récolter des données sur l'effet de la musique sur l'humeur et l'état mentale des gens.

Pour le traitement des formulaires

Nous allons sauvegarder certaines données transmise via les formulaires de manière totalement anonyme.
Les données sur l'identité sont utilisé exclusivement pour s'assurer pour qu'une même personne ne puissent répondre a un formulaire sur une courte période.


Tout notre travail est accessible gratuitement sur notre page GitHub
